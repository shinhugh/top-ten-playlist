Login:
curl http://localhost:8080/api/auth -v -H "Content-Type: application/json" -d '{"name":"dev","password":"dev"}'

Logout:
curl http://localhost:8080/api/auth -v -X DELETE

Create user account:
curl http://localhost:8080/api/user -v -H "Content-Type: application/json" -d '{"loginName":"dev","password":"dev","publicName":"Dev"}'

Read user account:
curl http://localhost:8080/api/user/dev -v -b "session="

Update user account:
curl http://localhost:8080/api/user/dev -v -X PUT -b "session=" -H "Content-Type: application/json" -d '{"loginName":"dev","password":"dev","publicName":"Dev"}'

Delete user account:
curl http://localhost:8080/api/user/dev -v -X DELETE -b "session="
